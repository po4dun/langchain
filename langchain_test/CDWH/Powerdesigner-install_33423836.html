DataWarehouse : Powerdesigner install

Created by Michael Hadorn, last modified by Florian Lukas Refardt on Feb. 16, 2023

All Software is downloaded here: I:\Verwaltung\ICT_CDWH\Software

  • Go to the SAP Data Services Download page, login with your SAP user, and download file: 51050743
  • Unzip "51050743.ZIP", execute "Autorun.exe" and choose to install the 64bit-version:
  • Choose English:
  • Click Next:
  • Choose License Method "Served" and click Next:
  • Enter the name of the license server (ictsapdaswe01) and the port (27000), then click Next:
  • Click Next in this dialogue:
  • Select "Switzerland(English)", agree to the terms of license and click Next:
  • Click Next:

    -- Check the features list and click Next:
  • Just click Next without selecting any user profiles:
  • Clickt Next:
  • Click Next:

  • Click Finish:

  • Go to the SAP Data Services Download page, login with your SAP user, and download the patch-file for SP07: PDVIEW166007_0-80001759.ZIP

  • Unzip the file and execute "PDES1667x64.6002.PL00.exe" to start the installation of the patch, click Next in the first dialogue:

  • Click Finish:

  • Configure the PD repository

  • Configure an ODBC driver PDREPO for the repository:

    • Driver: HDBODBC (must be installed with the HDB Client)
    • User: PDREPO

Configure the repository in PowerDesigner

  • Menu Repository -> Connect
  • Click on the setup button to setup the repository connection:

  • Enter PDREPO into the Repository Name filed and then click the OK button:

  • Click on the properties icon to specify the connection properties:

  • Repository name: PDREPO
  • Datasource name: the ODBC datasource previously configured
  • User name (HANA): PDREPO
  • Passoword (HANA): see KeePass

  • Click OK to connect to the repository
  • Repository name: PDREPO
  • Login Name (PowerDesigner): your personal login, i.e. weitzela
  • Password (PowerDesigner): your personal password. When logging in the first time you will be prompted to change the password. Please check with Arne for the initial password.

  • Before you can log in a user needs to be created for you in Power Designer

  • In order to make changes to the Repository (i.e. check in stuff) your user needs to be a member of the group CDWH_Developers
