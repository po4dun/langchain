IIS Installation and Configuration


After installing IIS and restarting the server. Start Master Data Services Configuration Manager

  • Click on Create Database



  • Enter Current User - Integrated Security, Test Connection and click "Next"



  • Name the Database (Default: MDS)




  • Specify Administrator Account




  • Finish Create Database
  • Start Sql Server Configuration Manager
  • Set SQL Server Agent Start Mode to Automatic and start SQL Server Agent



  • Start Web Configuration in Master Data Services Configuration Manager




  • Enter a Windows AD user for the the MDS Application Pool




  • Select MDS Database




Prepare SQL Server


  • Add the User Group MS\ICT_B_CDWH_MDS_Admins to the SQL Server
