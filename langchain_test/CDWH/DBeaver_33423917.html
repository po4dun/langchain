How to setup a database connection


After download from https://dbeaver.io/download/ and installation (possible without admin rights), run DBeaver and add a new connection by clicking on the symbol shown here


Enter "HANA" in the search field, highlight the SAP HANA Logo and continue (Weiter):

Now as first add the configuration to use an encrypted connection:

  • [1-4] enable encryption (to make sure, nobody can read your password while you connect)
    • encrypt = true
    • validateCertificate = false
  • [5] Host: ictthdbdwlp1.uhbs.ch
  • [6] Port: 33241
  • [7-8] Enter your Username and Initial-Password (Tick the box to save your password locally)
  • [9] Click to test the connection first
  • [10] Then click Finish (Fertigstellen)

  • Press OK if you get this dialogue:

You will then find the newly created connection in the database navigator: * Right click on the new connection and open a new SQL-Editor window:


Password change (via Webservice)

Changing passwords via SQL (below) might not work due to a problem with JDBC on HANA. Alternatively, the password can be change here:

For Prod access: https://ictthdbdwlp1.uhbs.ch:4332/sap/hana/xs/formLogin/login.html

For Kons access use: https://ictthdbdwlk1.uhbs.ch:4331/sap/hana/xs/formLogin/login.html

For Dev access use: https://ictthdbdwle1.uhbs.ch:4330/sap/hana/xs/formLogin/login.html

Password change (via SQL)

Enter: ALTER USER YourUsername PASSWORD "YourNewPassword" and execute the statement by clicking the "Play"-Button, as shown in the example below:

The successful password change should be confirmed with a green line:

Close the SQL-Editor window without saving and right-click again on your new connection, and edit the connection:

Enter your new password in the password field, test the connection and click OK:

The program notice the change in the connection settings and asks, whether you want to be connected freshly, that is a good idea (Ja):

When you open the tree in the database navigator below your new connection, you should now be able to see all the schemas and object where you have access to.

Congratulations, you are now connected to the CDWH!
