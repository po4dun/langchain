Application Information

Web-based clinical information system that reflects the complete cardiology process. The main focus of the application is the support of report creation, but also other procedures are supported, like triage, cardiac rehabilitation, assignment, and more. There are various interfaces to other clinical systems (i.e. Termdispo, ISMED) to ensure lean data entry.

Database Design

Data is stored accross five different databases, all located on the same server (COOSVC1 - COOSVC5). Each database just contains a part of the data. During the data load, the same tables from all 5 databases are merged together into one table, in order to get the full dataset.
