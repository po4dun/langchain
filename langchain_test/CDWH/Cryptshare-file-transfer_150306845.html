USB offers Cryptshare for sharing files (up to 4gb in size) within the hospital and to external parties (i.e. outside of the hospital).

Recipients receive a download link by email and need a password in order to access the data: This should be sent over an alternative channel like e.g. SMS or an alternative Email.

It can be accessed on Cryptshare secure file transfer and is self-explanatory.
