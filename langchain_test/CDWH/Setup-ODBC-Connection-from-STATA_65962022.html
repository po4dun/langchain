DataWarehouse : Setup ODBC Connection from STATA

Created by Fabian Franzeck, last modified on Nov. 04, 2022

Analyst might want to connect to CDWH from STATA. This can be done via ODBC (JDBC supported only from Version 17 on).

  1. You have to install & configure the ODBC datasource, in your operating system e.g. in Windows. The drivers are available from SAP Development Tools -> JDBC und ODBC Treiber. These are then available in Stata.

  2. From Stata use the ODBC commands (included in the standard Stata package) to interact with the database, e.g.

odbc load, ///
  exec("SELECT * FROM CDWH.V_IL_DIM_ORG_CUR;") ///
  clear lowercase noquote ///
  dsn("DWHP")             /// the name of your ODBC connection (e.g. in Windows)
  user("username")        /// your SAP HANA username - don't store in Stata scripts but e.g. as a global
  password("pw")          //  your SAP HANA password - don't store in Stata scripts but e.g. as a global
