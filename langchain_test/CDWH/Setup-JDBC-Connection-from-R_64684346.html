DataWarehouse : Setup JDBC Connection from R

Created by Fabian Franzeck, last modified on Juli 04, 2023

Many analyst wish to connect to CDWH directly from R, here is an R script template for setup of this connection.

The JDBC drivers are automatically installed on your computer when you setup a HANA connection with Dbeaver. The driver is a .jar file: just search for ngdbc.jar on your harddrive and copy the path to the definition of “jdbcDriver” in the R-script below). Alternatively, ngdbc.jar is available here: SAP Development Tools -> JDBC und ODBC Treiber.

# setup packages
# install.packages("RJDBC",dep=TRUE) 
library(RJDBC)

# set driver
jdbcDriver <- JDBC(driverClass="com.sap.db.jdbc.Driver",  
                   classPath="C:/Program Files/SAP/hdbclient/ngdbc.jar") # path to the JDBC Driver for SAP HANA as on your machine

# setup connection string
jdbcConnection <- dbConnect(jdbcDriver,
            "jdbc:sap://ictthdbdwlp1:33241/?autocommit=false"
            ,"username"   # your SAP HANA username, never store in R Scripts but externalise!
            ,"password",  # your SAP HANA password, never store in R Scripts but externalise!
            encrypt="true", # necessary for an encrypted connection to the database
            validateCertificate="false"  # necessary for an encrypted connection to the database
              )

# simple SQL query being written into a dataframe in R
DIM_ORG <- dbGetQuery(jdbcConnection, "SELECT * FROM CDWH.V_IL_DIM_ORG_CUR")
