DS Login

Login to the environment of your choice with Enterprise-Authentication, using your USB-Account, then click Log On:

DS DEV Repositories

There's a set of impersonal repositories only in the development environment, so that several developers can work in parallel. Each developer should "take" one of the available repositories from the list. The repositories starting with DS_CDWH_ are connected to the Oracle target platform, the one's starting with DSREPO_DEV_ are connected to the new HANA platform. Please keep this list uptodate:

  • DS_CDWH_01 / DSREPO_DEV_01: Michael Hadorn [hadornmi]
  • DS_CDWH_02 / DSREPO_DEV_02: Serena Bianco [biancose]
  • DS_CDWH_03 / DSREPO_DEV_03: Roger Mathis [mathisr]
  • DS_CDWH_04 / DSREPO_DEV_04: Arne Weitzel [weitzela]
  • DS_CDWH_05 / DSREPO_DEV_05: Devon Mahadevan [mahadevand]
  • DSREPO_DEV_06 (HANA only) : Richard Marti [martir]

The first time you log in you need to configure the Central Repository for your DEV environment. From Tools -> Central Repositories... Activate the central repository connection DS_CDWH_CENT as below.

Before you start developing, ensure that your repository contains the latest versions! To fetch the latest version from the central repository go to Tools -> Central Object Library, right click on the object you need and Get the latest version (with or without dependencies).
Test- and Prod-Environment has only one repository each, as there's no development allowed in those repositories.
