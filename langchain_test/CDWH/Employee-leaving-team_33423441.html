DataWarehouse : Employee leaving team

Created by Fabian Franzeck, last modified on Sept. 28, 2022

This page describes the steps necessary if a member of the team leaves

Bram should handle HR and they would inform the employee regarding giving back keys, etc. Rights for VD, etc are handled by IT. However, Employees can extend access to VD and Email easily, thus don't assume access will be immediately cancelled!

Computers/Laptops have to be given back to the department and can't be kept.


Steps for User management:

  1. SOS Helpdesktickets necessary:
    • Revoke access for I:\Verwaltung\ICT_CDWH (e.g. for keepass file)
    • ??? Revoke access to Emailfolder as described here: ICT-MSCCMailbox
  2. Remove User from gitlab groups cdwh, cdwh-objects, cdwh-generator
  3. Remove User rights from Confluence (any
  4. Remove User from Team Chats in Microsoft Teams
  5. Remove from fixed weekly meeting appointments in outlook
  6. Remove or deactivate personal User to SAP Hana database instances (via SAP Admin, e.g. Thierry Doppler)
  7. Remove User from Sharepoint (https://isp.usb.ch/teams/0219/)
