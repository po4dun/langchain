DataWarehouse : Add user to Confluence (Wiki)

Created by Fabian Franzeck, last modified on März 02, 2023

First the User has to be registered to confluence in the cloud with a licence (new employees are typically not automatically added), then the relevant user rights have to be granted.

  1. Open an ITSM Ticket: Ask for a Confluence Licence for that user (particularly for Fonds-Angestellte / Mitarbeiter ohne Lohn), but also verify for people with an D&ICT contract--> without licence, there’s no confluence access

  2. In the same ticket, ask for the user to be put in the confluence group: ICT_P_ictitwikilp01-Confluence-User

3. The user should now have access to (among other sites): https://usbch.atlassian.net/wiki/spaces/DFUA AND https://usbch.atlassian.net/wiki/spaces/CDWH

4. If not add in Berechtigungen for each Bereich necessary via the Bereichadmin
