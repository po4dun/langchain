DataWarehouse : Oracle install

Created by Michael Hadorn, last modified by Fabian Franzeck on Juni 08, 2022

  • Download
  • Windows 7
  • Windows 10

Download

  • Download and open the file Oracle.zip from sharepoint: Oracle.zip
  • Click "Extract", copy to location "C:\" and click OK:
  • Install ODBC Instant-Client for Oracle:
    Execute the file C:\Oracle\instantclient_12_2\odbc_install.exe with administrative rights:

Windows 7

Open Windows System Information Page:

  • Open System-Settings:
  • Open the Environment Variable Settings:
  • Set the User-Variable "Path" to C:\Oracle\instantclient_12_2\

  • Remove from System-Variable "Path": C:\Windows\Oracle\Client\x86\11.2.0.1.0\bin (else Toad uses the wrong version)
  • If User-Variable "TNS_ADMIN" already exists, change it to C:\Oracle\instantclient_12_2\network\admin , otherwise create it newly:
  • If System-Variable "ORACLE_HOME" already exists, change it to C:\Oracle\instantclient_12_2\ , otherwise create it newly:

Windows 10

  • Open the configuration for your personal environment variables:
  • Adapt the existing "Path" user variable and paste: C:\Oracle\instantclient_12_2\ at the beginning.
  • If not exists, create a new user variable "TNS_ADMIN" with the value: C:\Oracle\instantclient_12_2\network\admin\.
  • If not exists, create a new system variable "ORACLE_HOME" with the value: C:\Oracle\instantclient_12_2\.
  • At the end, your configuration of environment variables should look like:
