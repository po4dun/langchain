DataWarehouse : DBeaver CDWH-Project

Created by Richard Marti, last modified on Apr. 11, 2023

There is a CDWH-Project that can be imported in DBeaver. The Project contains all necessary Database-Connections to the HANA-Database and all relevant Source-System-Databases. All connections are already fully configured, what makes it obsolete to configure all connections manually and saves a lot of time.

DBeaver-Bug: TNS-Updates

DBeaver stores the configuration from the TNS-File together with the connection. When the TNS-File changes, DBeaver does not automatically read the updated TNS-Entry and the connection becomes invalid. If you encounter invalid Oracle-Connections in DBeaver, do the following:

  • Edit the connection and press the button ‚ÄúTest Connection‚Äù

  • This action ‚Äúforces‚Äù DBeaver to read the configuration from the TNS-File again

  • If the connection can be established successfully, click OK and done

  • If the connection cannot be established, you:

    • either have an outdated TNS-File (Check this article: Oracle install )

    • or the password has changed (Check KeePass to get the new password)
